Design System
Built to move fast.

This style guide documents all the tokens, components, and patterns that make up the Zeito Move web app. Use this as the single source of truth when building in Angular with Spartan UI.

Typography
Two fonts. Display for headings and labels; body for prose and inputs. Both from the Barlow family — athletic, condensed, purposeful.

Buttons
Four variants, three sizes. All use Barlow Condensed uppercase with letter-spacing. The primary action is always lime — never use it for more than one CTA per view.

Usage Rules
✓ Use Primary for the main action in a view — "Save & Schedule", "Sign In"
✓ Use Outline for secondary actions — "Save Draft", "Today", nav filters
✗ Never place two Primary buttons side by side — use Primary + Outline
